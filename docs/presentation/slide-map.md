## Notas De Edición

- Mantén el orden si la exposición sigue el guion original.
- Si se añade una diapositiva, actualiza el contador mental de secciones y este archivo.
- Si se elimina una diapositiva, revisa la progresión narrativa: problema, solución, arquitectura, dominio, operaciones y cierre.
